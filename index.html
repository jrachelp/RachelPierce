ABOUT

This is a website designed by Rachel Pierce which hosts all projects and activities completed in DSGN 234 Art of the Web at the University of Pennsylvania, taught by Nika Fisher. Fall of 2020.

PROJECTS

Compiled are all four projects completed during this course. These cover a range of coding practices, UI/UX applications, and a combination of visual and conceptual design.

  • Hello World

    An animated .gif expressing our individuality and introduction into the class.

  • 2020 Series

    A hand-coded website featuring a series of images describing our 2020 year.

  • Fanpage

    A contemporary website dedicated to something or someone we deem deserving of a fanpage.

  • Helpful App

    The design and conceptualization of a mobile application that is helpful to a larger audience than ourselves.

Activities

A collection of smaller activities worked on to develop technical skills, each having larger applications in the class and to our projects.

  • Coding From Life

    A practice in CSS, a still life portrait using only code.

  • Hovering

    Practice with CSS hovering functions.

  • CSS Animation

    A contemporary website dedicated to something or someone we deem deserving of a fanpage.

  • Navigation Review

    A critical review of various websites to gain a greater understanding of responsive and accessible navigation design.

  • Typesetting

    A practice in setting plain text into a typographic choice, responsive to both desktop and movile devices.

  • Media Queries

    Practice with responsive website design and the media query CSS functions.

  • jQuery 1

    An experiment with using jQuery functions.

  • jQuery 2

    An experiment with jQuery and representing time in a unique way.
